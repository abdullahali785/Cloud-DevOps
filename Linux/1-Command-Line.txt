Linux Command Line

1) Navigation

ls -> List files/folders
ls-l -> Detailed List
ls-a -> Includes hidden files

cd -> Change Directory
cd path/to/folders
cd ~ -> Home
cd .. -> One folder up

pwd -> Shows current directory


2) File Operations

cp -> Copy files
cp file1.txt file2.txt -> Copies file1 into file2
cp -r folder1 folder2 -> Recursive copy 

mv -> Move/rename files
mv old.txt new.txt
mv file.txt /path/to/destination/

rm -> Remove/delete files
rm file.txt
rm -r folder

mkdir -> New directory
touch -> New file


3) Viewing file contents

cat file.txt -> Prints whole file

less file.txt -> Scrollable view (for big files)

head file.txt -> First 10 lines
tail file.txt -> Last 10 lines


4) Text Processing

grep "pattern" file.txt -> Searches file.txt for "patter"

awk '{print $2}' file.txt -> Prints column 2 of file.txt

sed 's/old/new/g' file.txt -> Replaces all "old" with "new" in file.txt

Pipes '|' -> Sends output of one command into the next one

Redirects 
    > Overwrite 
    >> Append
